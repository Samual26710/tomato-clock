# 番茄钟 / tomato-clock

一个基于 Electron 的桌面番茄钟应用，提供倒计时、正计时、任务管理、专注统计、系统托盘和本地数据持久化，适合学习、写作、编码和日常专注工作。

## 功能特点

- 倒计时模式：默认 25 分钟，内置 25 / 50 / 5 分钟预设
- 正计时模式：适合不设上限的自由专注记录
- 任务清单：支持新建、编辑、删除、标记完成和选择当前任务
- 番茄统计：展示今日专注、今日番茄、本周累计和连续打卡天数
- 自动休息节奏：每完成 1 次专注进入休息，每完成 4 个完整番茄进入一次长休息
- 系统托盘：关闭窗口后仍可在托盘继续使用，支持开始、暂停、重置和重新打开窗口
- 本地持久化：任务、专注记录和设置会自动保存到本机

## 快速开始

### 直接运行打包版本

仓库中已经包含 Windows 打包产物，可以直接尝试：

- 安装版：`release/番茄钟-0.1.0-x64.exe`
- 便携版：`release-portable/番茄钟-0.1.0-x64-portable.exe`

### 从源码启动

先确保本机已安装 Node.js，然后在项目根目录执行：

```bash
npm install
npm start
```

### 打包应用

```bash
npm run pack
npm run dist:win
```

## 使用方式

### 1. 选择计时模式

- `倒计时`：适合标准番茄工作法
- `正计时`：适合开放式记录专注时长

### 2. 设置专注与休息时间

- 倒计时模式下可以直接点击 `25 分`、`50 分`、`5 分` 预设按钮
- 点击右上角 `设置` 可以修改：
  - 专注时长
  - 短休息时长
  - 长休息时长
  - 专注结束后是否自动开始休息

### 3. 创建并选择任务

- 点击 `+ 新建` 创建任务
- 填写任务标题、预估番茄数和备注
- 点击任务项即可将其设为当前任务
- 已完成任务可以勾选标记，不再继续作为当前专注任务

### 4. 开始一次专注

- 点击 `开始` 启动计时
- 运行中可以 `暂停`
- 暂停后可以 `继续`
- 点击 `重置` 可以清空当前计时
- 如果当前处于休息阶段，点击 `跳过休息` 会直接回到下一轮专注

### 5. 专注完成后的行为

在倒计时模式下，一次专注结束后，应用会自动：

- 记录一条专注会话
- 将本次专注时长累计到当前任务
- 更新统计面板
- 发送系统通知
- 切换到短休息或长休息

每完成 4 个完整番茄，会进入一次长休息。

### 6. 查看统计数据

首页会持续展示以下信息：

- 今日专注时长
- 今日完成番茄数
- 本周累计专注时长
- 连续打卡天数

### 7. 使用系统托盘

- 关闭主窗口不会退出应用，而是隐藏到系统托盘
- 点击托盘图标可以重新打开主窗口
- 托盘右键菜单支持：
  - 显示主窗口
  - 开始 / 继续
  - 暂停计时
  - 重置计时
  - 退出应用

## 数据保存位置

应用会把数据保存到系统 `userData` 目录中的 `data.json`。

Windows 常见位置：

```text
%APPDATA%/pomodoro-desktop/data.json
```

保存内容包括：

- 任务列表
- 当前激活任务
- 专注记录
- 专注与休息设置
